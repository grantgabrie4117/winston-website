Introduction

This project analyzes the BGSU Esports Overwatch Teams performance over the 2025 season using match statistics. The dataset includes per-player metrics such as eliminations, assists, deaths, damage, and healing, across multiple maps and opponents.

Data Description

# Load Data
data <- read.csv("Overwatch Season Stat Tracker(Raw Match Data) (3).csv", stringsAsFactors = FALSE)

# Clean numeric columns
data <- data %>%
  mutate(across(c(Damage.Dealt, Healing.Done, Damage.Mitigated),
                ~ as.numeric(gsub(",", "", .))))

# Remove empty columns
data <- data %>% select(-c(Final.Blows, Ultimates.Used, Ultimates.Earned))

Purpose

The purpose of this analysis is to evaluate the performance trends of the BGSU Overwatch team. Specifically, the analysis seeks to:

  • Examine distributions of core performance stats such as K/D ratio and healing done.
  • Explore relationships between performance metrics and match outcomes.
  • Compare different players and heroes to identify strengths and patterns.

Data Analysis

1. Distribution of Key Variables

data <- data %>% mutate(KD.Ratio = Eliminations / ifelse(Deaths == 0, 1, Deaths))

ggplot(data, aes(x = KD.Ratio)) +
  geom_histogram(binwidth = 0.25, fill = "steelblue", color = "white") +
  labs(title = "Distribution of K/D Ratios", x = "K/D Ratio", y = "Count")

Conclusions

From the analysis, we can observe performance patterns across players and heroes. Higher damage and healing values generally correspond to stronger team outcomes. Future work could include player-by-player performance tracking and deeper hero-specific analysis.
